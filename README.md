# 📱 Pokedex Mobile-First - Vanilla JavaScript

A modern, responsive Pokedex built with Vanilla HTML5, CSS3, and JavaScript. This project focuses on high performance, clean code, and the use of modern web standards like the `<dialog>` element for modal management.

## 🚀 Features

- **Dynamic Loading:** Pokemons are fetched from the PokeAPI with pagination.
- **Detailed Popups:** Uses the native `<dialog>` tag to show specific stats, height, weight, and abilities for each Pokemon.
- **Responsive Design:** Optimized for mobile devices with a "Pop-out" image effect.
- **Theme per Type:** The modal background dynamically changes color based on the primary type of the Pokemon.
- **High Performance:** Implements Event Delegation to handle clicks efficiently across the list.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and `<dialog>` element.
- **CSS3:** Custom properties (variables), Flexbox, Grid, and Backdrop filters.
- **JavaScript (Vanilla):** Fetch API, DOM manipulation, and Event Delegation.
- **PokeAPI:** Source for all Pokemon data.
